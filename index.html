My CSS Selectors Cheat Sheet

Here is a table of common CSS Selectors & Combinator Selectors

Common CSS Selectors

Selector Example Description
#id #myname Selects the element with id="myname"
.class .intro Selects all elements with class="intro"
element.class p.intro Selects only <p> elements with class="intro"
( * ) * The universal selector (*) selects all HTML elements on the page.
Element p Selects all <p> elements/td>
element,element div, p Selects all <div> elements and all <p> elements

CSS Combinator Selectors

Selector Example Description
element element div p Selects all <p> elements inside <div> elements
element > element div > p Selects all <p> elements where the parent is a <div> element
element + element div + p Selects the first <p> element that are placed immediately after <div> elements
element1~element2 p ~ ul Selects every <<l> element that are preceded by a <p> element
